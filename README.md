# ⚔ Commander Companion

A free, offline-capable MTG Commander companion app for new players.

**Live app:** https://YOUR-USERNAME.github.io/mtg-commander

## Features
- 📋 **Turn Guide** — Full phase-by-phase breakdown with priority indicators
- ❤️ **Life Tracker** — Up to 6 players, commander damage, poison counters, floating mana pool
- ⚡ **Interactions** — 12 keyword interaction rulings with searchable database
- 📖 **Keywords** — 20 keyword glossary with difficulty ratings and examples
- 📴 **Works fully offline** — install to home screen, no internet needed after first load

## Install on iPhone
1. Open the link in Safari
2. Tap the Share button
3. Tap "Add to Home Screen"
4. Done — works offline forever after

## Install on Android
1. Open the link in Chrome
2. Tap the three dots menu
3. Tap "Add to Home Screen" or "Install App"

## Rules accuracy
All rulings cite official Comprehensive Rules section numbers. For tournament disputes always verify at [rules.wizards.com](https://rules.wizards.com).
